# Гайд-онбординг: Менеджер

Роль **Менеджер** отвечает за работу с клиентами: от первого обращения до подписания договора и передачи проекта в производство. Вы видите только **своих** клиентов, договоры и проекты — данные других менеджеров скрыты.

---

## Меню системы

После входа вы увидите в боковом меню следующие разделы:

| Раздел | Описание |
|--------|---------|
| Главная | Дашборд с приветствием |
| Клиенты | Ваша воронка продаж |
| Договоры | Ваши договоры |
| Календарь | Годовой вид ваших мероприятий |
| Проекты | Ваши проекты |
| Производство | Обзор задач и закупок по вашим проектам |
| Дизайнер | Макеты (просмотр) |
| Менеджер | Ваши личные задачи и расходы |
| Инвентарь | Склад: реквизит и ткани |
| Инфо* | Персонал и подрядчики |
| Почта | Входящие заявки с сайта |
| Статистика | Ваша личная аналитика |
| История | Журнал ваших действий |
| Сообщения | Общий командный чат |

> *Раздел "Инфо" виден только если руководитель включил вам флаг "Доступ к разделу Инфо".

---

## 1. Вход в систему

1. Перейдите по адресу системы (например, `https://crm.maketdecor.ru`)
2. Введите логин и пароль
3. Нажмите **"Войти"**

---

## 2. Главная (Дашборд)

После входа вы попадаете на дашборд с приветствием.

- **Уведомления** (колокольчик в шапке) — оповещения о новых заявках, изменениях статусов. Можно отметить прочитанными по одному или все сразу.
- **Глобальный поиск** — поле в шапке. Ищет по вашим клиентам, договорам и проектам (имя, площадка, номер счёта). Минимум 2 символа.
- **Смена обоев** — кнопка в шапке, выбор из 15 вариантов фона.

---

## 3. Клиенты — воронка продаж

Раздел **"Клиенты"** — ваше основное рабочее место.

### Статусы воронки

```
Встреча → Обсуждение → Смета → Договор
                              ↘
                            Отказ
```

| Статус | Когда использовать |
|--------|-------------------|
| Встреча | Первый контакт, встреча назначена |
| Обсуждение | Встреча прошла, обсуждаем детали |
| Смета | Готовим/отправили смету |
| Договор | Договор подписан |
| Отказ | Клиент отказался |

### Создание нового клиента

1. Нажмите **"Добавить клиента"**
2. Заполните форму:
   - **Имя клиента** — ФИО или название компании
   - **Телефон, Email** — контактные данные
   - **Дата обращения** — когда поступил запрос
   - **Дата встречи** — планируемое время встречи
   - **Дата мероприятия** — когда пройдёт событие
   - **Площадка** — место проведения
   - **Тип мероприятия**: Свадьба, Корпоратив, День рождения, Другое
   - **Источник** — откуда узнал клиент
   - **Описание идеи** — что хочет клиент
3. Нажмите **"Сохранить"**

### Просмотр и фильтрация

- Таблица с сортировкой по любой колонке
- Вкладки: **Активные клиенты** / **Отказы**
- Фильтры: по статусу, типу мероприятия
- Поиск по имени/площадке

### Обновление статуса

Откройте карточку клиента → измените статус → изменение сохраняется.

### Конвертация клиента в договор

1. Откройте карточку клиента → **"Конвертировать в договор"**
2. Система автоматически:
   - Создаёт запись в "Договорах" со всеми данными клиента
   - Переносит прикреплённые сметы
   - Обновляет статус клиента на `Договор`

### Отклонение клиента

1. **"Отклонить"** → введите причину (обязательно) → **"Подтвердить"**
2. Клиент переходит в `Отказ` и учитывается в аналитике

---

## 4. Работа со сметами

Сметы прикрепляются к клиенту и переносятся в договор при конвертации.

### Загрузка сметы

1. Откройте карточку клиента → вкладка **"Файлы"**
2. **"Загрузить файл"** → выберите файл (Excel, PDF)
3. Укажите **версию** (1, 2, 3...)
4. При необходимости — отметьте **"Командная версия"** (без цен, для команды)

### Версии смет

- Версия 1, 2, 3... — последовательные редакции
- Каждая версия хранится отдельно, можно скачать любую
- Командная версия — копия без цен для производства/дизайнера

---

## 5. Договоры

Раздел **"Договоры"** — ваши договоры, созданные из клиентов.

### Карточка договора

- Данные клиента, даты мероприятия и подписания
- **Сумма договора**, предоплата, **номер счёта**
- **Статус макета** — отслеживание работы дизайнера:

| Статус макета | Значение |
|---------------|---------|
| Ожидает | Ожидает начала работы |
| В работе | Дизайнер работает |
| Согласован | Макет одобрен клиентом |
| Передан | Передан в производство |
| Отменён | Отменён |

### Файлы договора

1. Откройте договор → вкладка **"Файлы"**
2. Загрузите файл договора (Word/PDF) и/или счёт
3. Укажите тип файла при загрузке

### Референсные изображения

1. Вкладка **"Референсы"** → загрузите фото-вдохновения от клиента
2. Дизайнер сможет просмотреть их при создании макета

---

## 6. Проекты

Раздел **"Проекты"** — производственные карточки ваших мероприятий.

### Создание проекта

1. **"Создать проект"** → заполните:
   - Название, дата мероприятия, площадка
   - Описание — технические детали для производства
   - Цвет — метка в календаре
   - Привязка к договору (опционально)
2. Нажмите **"Сохранить"**

### Задачи производства

Чеклист того, что нужно сделать производству:

1. Откройте проект → вкладка **"Задачи"**
2. **"Добавить задачу"** → введите название → сохраните
3. Прогресс виден как N/M (например, 4/6 выполнено)
4. Когда производство отмечает задачу — видно кто и когда

### Закупки

Список материалов для проекта:

1. Вкладка **"Закупки"** → **"Добавить позицию"**
2. Введите название, количество
3. Позиции отмечаются как купленные/не купленные

### Чат проекта

1. Вкладка **"Чат"** → введите сообщение → Enter
2. Сообщения доставляются в реальном времени всем участникам (менеджер + руководитель + производство)

### Фотографии проекта

Два типа изображений:
- **Эскизы заказа** — референсы, схемы
- **Фото производства** — фотографии сборки, готовых изделий

---

## 7. Производство (обзор)

Раздел **"Производство"** — сводный обзор задач и закупок по вашим проектам.

- Для каждого проекта: название, дата, прогресс задач, прогресс закупок
- Чекбоксы для быстрой отметки выполнения
- Используйте для быстрого контроля готовности без перехода в каждый проект

---

## 8. Календарь мероприятий

Раздел **"Календарь"** — годовой вид ваших мероприятий.

- Сетка по неделям (Пн-Вс), навигация по годам
- Проекты отображаются цветными метками:
  - Оранжевый — монтаж
  - Синий — демонтаж
  - Зелёный — бронь
- Нажмите на метку → переход к карточке проекта

---

## 9. Макеты дизайнера

Раздел **"Дизайнер"** — просмотр макетов.

- Карточки макетов: номер, даты, сложность (1-5 звёзд), статус
- Загруженные изображения по зонам (церемония, президиум, банкетный зал и т.д.)
- Менеджер работает **в режиме просмотра** — создание и редактирование выполняет дизайнер

---

## 10. Личные задачи и расходы

Раздел **"Менеджер"** — ваш персональный органайзер.

### Личные задачи

1. **"Добавить задачу"** → описание, срок выполнения → сохраните
2. Отметьте выполненную задачу чекбоксом
3. Задачи видны только вам (и руководителю)

### Расходы

1. **"Добавить расход"** → сумма, описание, дата, категория (общие / расходники / проектные)
2. При необходимости — привяжите к проекту
3. Сохраните

---

## 11. Инвентарь

Раздел **"Инвентарь"** — учёт реквизита и тканей на складе.

### Реквизит

- Дерево категорий с позициями (название, цвет, количество)
- Поиск по названию/цвету
- Журнал повреждений

### Ткани

- Ссылка из раздела "Инвентарь"
- Материал, цвет, ширина, раскрой, длина, поставщик

Используйте для проверки наличия реквизита при планировании мероприятий.

## 12. Персонал и подрядчики

Раздел **"Инфо"** — доступен только если руководитель включил вам **"Доступ к разделу Инфо"**.

- **Персонал**: штатные сотрудники, фрилансеры, водители — контакты и данные
- **Подрядчики**: поставщики по категориям — компания, телефон, адрес

---

## 13. Почта (входящие заявки)

Раздел **"Почта"** — заявки с формы на сайте.

- Список заявок: отправитель, тема, текст, дата, статус прочтения
- Счётчик непрочитанных
- **"Создать клиента"** — данные заявки перенесутся в форму нового клиента

---

## 14. Статистика

Раздел **"Статистика"** — ваши личные показатели.

| Метрика | Описание |
|---------|---------|
| Всего клиентов | Ваши активные клиенты |
| Всего договоров | Ваши договоры |
| Всего проектов | Ваши проекты |
| Конверсия | % клиентов, дошедших до договора |
| Воронка продаж | Распределение по статусам |
| Выручка по месяцам | Суммы ваших договоров |

Данные отображаются только по **вашим** клиентам.

## 15. История изменений

Раздел **"История"** — журнал **ваших** действий в системе.

- Создание и редактирование клиентов, договоров, проектов
- Смена статусов, действия с файлами
- Фильтр по типу сущности
- Пагинация

## 16. Общий чат

Раздел **"Сообщения"** — общий командный чат.

- Все сотрудники видят одну ленту сообщений в реальном времени
- Используйте для общих объявлений и координации

> Для обсуждения конкретного проекта используйте **чат проекта** (Проекты → карточка → вкладка "Чат").

## Гайд-онбординг: первые шаги в системе

Пошаговое руководство для нового менеджера — от первого входа до полного рабочего цикла с клиентом.

### Шаг 1. Первый вход и знакомство

1. Откройте `https://crm.maketdecor.ru`, введите логин и пароль от руководителя
2. Вы на **Дашборде**. Слева — боковое меню с вашими разделами
3. Нажмите колокольчик — проверьте уведомления
4. Смените обои на понравившиеся (кнопка в шапке)
5. Откройте **"Сообщения"** — поздоровайтесь с командой

### Шаг 2. Проверьте входящие заявки

1. Откройте **"Почта"** — посмотрите, есть ли новые заявки с сайта
2. Если есть подходящая заявка — нажмите **"Создать клиента"**
3. Данные заявки автоматически подставятся в форму

### Шаг 3. Создайте первого клиента

1. **"Клиенты" → "Добавить клиента"**
2. Заполните: имя, контакты, дату мероприятия, площадку, тип, описание идеи
3. Сохраните — клиент появится со статусом `Встреча`
4. Создайте себе напоминание: **"Менеджер" → "Добавить задачу"** → "Подготовить прайс для клиента"

### Шаг 4. Проведите клиента по воронке

1. После встречи — откройте карточку клиента, смените статус на `Обсуждение`
2. Подготовьте смету → загрузите файл во вкладку **"Файлы"** (версия 1)
3. Загрузите командную версию без цен
4. Смените статус на `Смета`
5. При согласовании — нажмите **"Конвертировать в договор"**

### Шаг 5. Оформите договор

1. Перейдите в **"Договоры"** → откройте новый договор
2. Заполните сумму, предоплату, номер счёта, дату подписания
3. Загрузите подписанный договор и счёт во вкладку **"Файлы"**
4. Загрузите референсные изображения от клиента во вкладку **"Референсы"**
5. Напишите дизайнеру в **"Сообщения"**: "Договор подписан, референсы залиты"

### Шаг 6. Создайте проект

1. **"Проекты" → "Создать проект"** → заполните детали, привяжите к договору
2. Добавьте **задачи** для производства: "Сборка арки", "Декор президиума"...
3. Добавьте **закупки**: "Пионы 60 шт.", "Свечи 30 шт."...
4. Загрузите **эскизы** во вкладку "Фотографии"
5. Напишите в **чат проекта**: "Монтаж такого-то числа, доступ через служебный вход"

### Шаг 7. Контролируйте подготовку

1. Следите за прогрессом задач и закупок в проекте (или в разделе **"Производство"**)
2. В **"Дизайнер"** проверяйте статус макета
3. В **"Календарь"** смотрите плотность мероприятий
4. Уточняйте детали в **чате проекта**

### Шаг 8. После мероприятия

1. Убедитесь, что все задачи в проекте закрыты
2. Загрузите финальные фото в **"Фотографии производства"**
3. Внесите расходы: **"Менеджер" → "Добавить расход"**
4. Проверьте свою **"Статистику"** — как обновилась воронка и выручка

## Ежедневный маршрут менеджера

> **Утро**: Уведомления → Сообщения → Почта (новые заявки)
>
> **День**: Клиенты (обновление статусов) → Договоры (контроль) → Проекты (прогресс задач) → Дизайнер (статус макета)
>
> **Конец дня**: Менеджер (задачи на завтра) → Статистика (результаты)

## Типичные рабочие сценарии

### Сценарий 1: Обработка новой заявки с сайта

1. **"Почта"** → найти новую заявку → **"Создать клиента"**
2. Дополнить контакты, назначить дату встречи → сохранить

### Сценарий 2: Ведение клиента от встречи до договора

1. Создать клиента → `Встреча`
2. После встречи → `Обсуждение`
3. Загрузить смету → `Смета`
4. **"Конвертировать в договор"** → заполнить сумму, файлы

### Сценарий 3: Подготовка проекта для производства

1. Создать проект по договору
2. Добавить задачи и закупки
3. Загрузить эскизы
4. Написать в чат проекта

### Сценарий 4: Контроль готовности

1. Открыть проект → проверить прогресс задач (N/M)
2. Раздел **"Производство"** — сводный обзор
3. Уточнить в чате проекта
