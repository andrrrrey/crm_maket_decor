# Гайд-онбординг: Производство

Роль **Производство** — сотрудники, которые изготавливают и монтируют декор. Вы видите задачи по предстоящим мероприятиям, управляете складом и ведёте журнал выполненных работ.

**Ключевое ограничение**: вы видите проекты только в **месяцах, открытых руководителем**. Пока руководитель не откроет месяц — проекты в нём не отображаются.

---

## Меню системы

После входа вы увидите в боковом меню **6 разделов**:

| Раздел | Описание |
|--------|---------|
| Главная | Дашборд с приветствием |
| Календарь | Годовой вид мероприятий (только открытые месяцы) |
| Производство | Обзор проектов с задачами и закупками |
| Инвентарь | Склад: реквизит и ткани |
| История | Журнал ваших действий |
| Сообщения | Общий командный чат |

Разделы клиентов, договоров, дизайнера, статистики и настроек скрыты — они не нужны для производственной работы.

---

## 1. Вход в систему

1. Перейдите по адресу системы (например, `https://crm.maketdecor.ru`)
2. Введите логин и пароль
3. Нажмите **"Войти"**

---

## 2. Главная (Дашборд)

После входа вы попадаете на дашборд с приветствием.

- **Уведомления** (колокольчик в шапке) — оповещения о новых задачах и сообщениях. Можно отметить прочитанными по одному или все сразу.
- **Смена обоев** — кнопка в шапке, выбор из 15 вариантов фона.

---

## 3. Производство — основная рабочая область

Раздел **"Производство"** — сводный обзор всех проектов с задачами и закупками.

### Почему проекты могут не отображаться

Вы видите проекты только в **открытых месяцах**. Если список пуст — руководитель ещё не открыл текущий месяц. Уточните у руководителя в общем чате.

### Что показывает раздел

Для каждого проекта отображается:
- **Название** с цветовой меткой
- **Дата** мероприятия и **менеджер**
- **Задачи** — полоса прогресса (выполнено / всего) + чекбоксы по каждой задаче
- **Закупки** — полоса прогресса + чекбоксы

### Выполнение задач

1. Найдите нужный проект в списке
2. В блоке **"Задачи"** нажмите на **чекбокс** нужной задачи
3. Задача отмечается выполненной — система записывает **ваше имя** и **дату/время**
4. Прогресс обновляется (полоса прогресса, счётчик N/M)

> Отмена: повторно нажмите на чекбокс, чтобы снять отметку.

### Отметка закупок

1. В блоке **"Закупки"** нажмите чекбокс напротив купленной позиции
2. Статус обновляется для всей команды

### Переход к полной карточке проекта

Нажмите на название проекта — откроется полная карточка с вкладками:
- **Описание** — технические детали, площадка, время монтажа
- **Задачи** — полный чеклист с подробностями
- **Закупки** — полный список материалов
- **Чат** — общение с менеджером и руководителем
- **Фотографии** — эскизы заказа и фото производства

---

## 4. Чат проекта

У каждого проекта есть внутренний чат для команды.

1. Откройте карточку проекта → вкладка **"Чат"**
2. Введите сообщение → **Enter** или кнопка отправки
3. Сообщения доставляются в реальном времени (Socket.io)

**Кто видит чат проекта**: менеджер, руководитель и все сотрудники производства.

**Используйте для**:
- Вопросов по техническому заданию
- Уточнения деталей у менеджера
- Сообщений о готовности этапов
- Координации с коллегами

---

## 5. Фотографии проекта

В карточке проекта → вкладка **"Фотографии"**:

| Тип | Описание |
|-----|---------|
| Эскизы заказа | Рисунки, схемы, референсы от менеджера |
| Фото производства | Фотографии готовых изделий, сборки, монтажа |

**Загрузка фото производства**:
1. Выберите тип "Производство"
2. Нажмите область загрузки или перетащите файлы
3. Форматы: JPG, PNG, WebP (до 50 МБ)

---

## 6. Календарь мероприятий

Раздел **"Календарь"** — годовой вид мероприятий в открытых месяцах.

- Сетка по неделям (Пн-Вс), навигация по годам
- Проекты отображаются цветными метками:
  - Оранжевый — монтаж
  - Синий — демонтаж
  - Зелёный — бронь
- Нажмите на метку → переход к карточке проекта
- Закрытые месяцы не отображают проекты

**Используйте для**: планирования нагрузки, просмотра ближайших мероприятий, понимания очерёдности работ.

---

## 7. Склад — инвентарь

Раздел **"Инвентарь"** — учёт реквизита и оборудования.

### Структура

Инвентарь организован в **дерево категорий** (Арки, Вазы, Подсвечники, Текстиль, Оборудование и т.д.). Раскрывайте категории для просмотра позиций.

### Просмотр позиции

- Название, цвет, количество на складе
- Поиск по названию/цвету

### Добавление и редактирование позиций

1. **"Добавить позицию"** в нужной категории
2. Заполните: название, цвет, количество
3. Сохраните

### Обновление количества

Если взяли реквизит на мероприятие или вернули:
1. Откройте позицию → **"Редактировать"**
2. Измените количество → сохраните

### Фиксация повреждения

1. Найдите позицию → **"Зафиксировать повреждение"**
2. Описание повреждения, дата, количество повреждённых единиц
3. **"Сохранить"** — запись попадает в журнал, видна руководителю

> **Важно**: фиксируйте повреждения сразу — это помогает своевременно заменять реквизит.

---

## 8. Склад — ткани

Раздел **"Ткани"** (ссылка из раздела "Инвентарь") — отдельный учёт текстиля.

- Материал, цвет, ширина (см), раскрой, общая длина (м)
- Поставщик, год покупки, примечания
- Группировка по типу материала
- Добавление и редактирование позиций

---

## 9. История изменений

Раздел **"История"** — журнал **ваших** действий в системе.

Фиксируется:
- Какие задачи отмечены выполненными (с датой и временем)
- Изменения в инвентаре
- Зафиксированные повреждения
- Другие действия

Используйте для подтверждения выполненной работы и проверки дат.

---

## 10. Общий чат

Раздел **"Сообщения"** — командный чат для всей студии.

1. Откройте "Сообщения" → введите сообщение → Enter
2. Сообщения в реальном времени, видны всем ролям

**Отличие от чата проекта**: общий чат — для общих вопросов и объявлений; чат проекта — для обсуждения конкретного мероприятия.

## Гайд-онбординг: первые шаги в системе

Пошаговое руководство для нового сотрудника производства — от первого входа до полного рабочего цикла.

### Шаг 1. Первый вход

1. Откройте `https://crm.maketdecor.ru`, введите логин и пароль от руководителя
2. Вы на **Дашборде**. Слева — 6 разделов: Главная, Календарь, Производство, Инвентарь, История, Сообщения
3. Нажмите колокольчик — проверьте уведомления
4. Смените обои на понравившиеся

### Шаг 2. Познакомьтесь с командой

1. Откройте **"Сообщения"** — прочитайте последние сообщения
2. Представьтесь команде в общем чате

### Шаг 3. Посмотрите предстоящие мероприятия

1. Откройте **"Календарь"** — годовой вид с цветными метками
2. Найдите ближайшие мероприятия в текущем и следующем месяцах
3. Нажмите на метку — перейдёте к карточке проекта
4. Если календарь пуст — руководитель ещё не открыл месяц, напишите ему в чат

### Шаг 4. Изучите текущие проекты

1. Откройте **"Производство"** — сводный обзор всех проектов
2. Для каждого проекта видны задачи (что делать) и закупки (что купить)
3. Нажмите на название проекта → полная карточка:
   - Прочитайте **описание** — площадка, время, особенности
   - Изучите **эскизы** на вкладке "Фотографии" — что именно нужно делать
   - Проверьте **закупки** — какие материалы нужны
   - Загляните в **чат** — возможно, менеджер оставил важные детали

### Шаг 5. Ознакомьтесь со складом

1. Откройте **"Инвентарь"** — дерево категорий с позициями
2. Изучите, какой реквизит есть в наличии
3. Запомните структуру категорий
4. Перейдите в **"Ткани"** — проверьте наличие текстиля

### Шаг 6. Начните работать с задачами

1. В разделе **"Производство"** найдите ближайший проект
2. Читайте описание задачи — что конкретно нужно сделать
3. По мере выполнения — нажимайте чекбоксы
4. Система запишет ваше имя и время выполнения
5. Если что-то непонятно — напишите в **чат проекта**

### Шаг 7. Подготовка к мероприятию

1. За несколько дней проверьте прогресс задач и закупок
2. Сверьте наличие реквизита в **"Инвентаре"**
3. Если чего-то не хватает — напишите в чат проекта
4. В день мероприятия — отмечайте задачи по мере выполнения (монтаж, финальная проверка)
5. Сделайте фото готовой площадки → загрузите в **"Фотографии производства"**

### Шаг 8. После мероприятия

1. Откройте **"Инвентарь"** → обновите количество возвращённых позиций
2. Если что-то повреждено — **"Зафиксировать повреждение"** с описанием
3. Аналогично в **"Тканях"** — обновите остатки
4. Проверьте свои действия в **"Истории"**

---

## Ежедневный маршрут производства

> **Утро**: Уведомления → Производство (задачи на сегодня) → Календарь (ближайшие мероприятия)
>
> **День**: Выполнение задач (чекбоксы) → Чат проекта (координация) → Инвентарь (взять/вернуть реквизит)
>
> **Вечер**: Загрузка фото → Обновление инвентаря → Фиксация повреждений

---

## Типичные рабочие сценарии

### Сценарий 1: Начало рабочего дня

1. **"Производство"** → посмотреть ближайшие проекты
2. Найти задачи на сегодня
3. При вопросах — написать в **чат проекта**

### Сценарий 2: Выполнение задач

1. **"Производство"** → нужный проект
2. Последовательно отмечать задачи чекбоксами
3. Сделать фото результатов → загрузить в проект

### Сценарий 3: Получение материалов

1. **"Инвентарь"** → найти нужные позиции
2. Проверить наличие
3. Обновить количество после получения

### Сценарий 4: Возврат после мероприятия

1. **"Инвентарь"** → найти использованные позиции
2. Обновить количество (прибавить возвращённое)
3. Если повреждено → **"Зафиксировать повреждение"**

### Сценарий 5: Не вижу проекты

1. Руководитель не открыл месяц
2. Напишите в **"Сообщения"** руководителю

---

## Частые вопросы

**Почему список проектов пуст?**
Руководитель ещё не открыл текущий месяц. Уточните в общем чате.

**Как отменить отметку задачи?**
Повторно нажмите на чекбокс — отметка снимается.

**Как сообщить о повреждении?**
Инвентарь → позиция → "Зафиксировать повреждение" → описание + дата.

**Не вижу нужный проект — что делать?**
Проверьте, открыт ли месяц мероприятия. Если нет — напишите руководителю.
