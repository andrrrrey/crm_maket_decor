# Инструкция: Дизайнер

Роль **Дизайнер** — создание и ведение макетов оформления мероприятий. Дизайнер работает с визуальными материалами: загружает изображения по зонам декора, отслеживает статус согласования с клиентами, общается с командой.

Дизайнер видит только **свои** макеты. Доступ к клиентам, договорам и проектам ограничен.

## 1. Вход в систему

1. Перейдите по адресу системы (например, `https://crm.maketdecor.ru`)
2. Введите логин и пароль
3. Нажмите **«Войти»**

В боковом меню отображаются только разделы, доступные дизайнеру:
- Дашборд
- Дизайнер (макеты)
- Сообщения
- История

Разделы клиентов, договоров, производства и склада скрыты — они не входят в область работы дизайнера.

---

## 2. Главная (Дашборд)

После входа вы попадаете на дашборд.

**Что здесь есть:**
- Фоновое изображение
- Уведомления (колокольчик в шапке) — оповещения о новых задачах

**Смена фона:**
1. Нажмите иконку смены обоев
2. Выберите понравившееся изображение из галереи (15 вариантов)

## 3. Макеты — основная рабочая область

Раздел **«Дизайнер»** — список всех ваших макетов.

### Что такое макет

Макет — это набор дизайн-материалов для одного мероприятия. Он включает:
- Метаданные: месяц, даты, сложность
- Изображения по зонам оформления (церемония, президиум и т.д.)
- Статус согласования

### Список макетов

Таблица показывает:
- Месяц мероприятия
- Даты установки и монтажа
- Сложность (1–5)
- Текущий статус
- Дата создания

Нажмите на строку, чтобы открыть полную карточку макета.

---

## 4. Создание нового макета

1. Нажмите кнопку **«Создать макет»** (правый верхний угол раздела)
2. Заполните форму:

| Поле | Описание |
|------|---------|
| **Месяц** | Месяц проведения мероприятия |
| **Дата начала (старт)** | Когда начинаются работы по декору |
| **Дата монтажа** | Дата финального монтажа на площадке |
| **Сложность** | Оценка от 1 до 5 (1 — простое, 5 — очень сложное) |
| **Количество дней** | Сколько дней займёт подготовка |
| **Описание** | Дополнительные заметки по проекту |

3. Нажмите **«Сохранить»**

Макет создаётся со статусом «В работе» (или начальным статусом по умолчанию).

---

## 5. Редактирование макета

1. Откройте нужный макет из списка
2. Нажмите **«Редактировать»**
3. Измените нужные поля
4. Нажмите **«Сохранить»**

**Что можно редактировать:**
- Все метаданные (месяц, даты, сложность, описание)
- Статус макета
- Загружать и удалять изображения

---

## 6. Загрузка изображений по зонам

Ключевая функция — загрузка дизайн-изображений для каждой зоны декора.

### Зоны оформления

Каждый макет может содержать изображения по нескольким зонам:

| Зона | Описание |
|------|---------|
| Церемония | Оформление церемониальной зоны |
| Президиум | Зона для молодожёнов / почётных гостей |
| Банкетный зал | Общее оформление зала |
| Фотозона | Декоративный фон для фотографий |
| Вход | Входная группа |
| Другое | Прочие зоны по запросу |

### Загрузка изображений

1. Откройте карточку макета
2. Найдите блок с нужной зоной
3. Нажмите на область загрузки или кнопку **«Добавить изображение»**
4. Выберите файл(ы) с компьютера
5. Поддерживаемые форматы: JPG, PNG, WebP
6. Максимальный размер файла: 50 МБ

### Удаление изображения

1. Наведите курсор на изображение
2. Нажмите иконку удаления (корзина / крестик)
3. Подтвердите удаление

### Просмотр в полном размере

Нажмите на миниатюру изображения, чтобы открыть его в полном размере.

## 7. Статусы макета

Статус показывает, на каком этапе находится макет.

### Доступные статусы

| Статус | Значение |
|--------|---------|
| В работе | Дизайнер активно работает над макетом |
| На согласовании | Макет отправлен клиенту на согласование |
| Согласован | Клиент одобрил макет |
| Передан в производство | Макет передан команде производства |
| Закрыт | Работа по макету завершена |

### Смена статуса

1. Откройте карточку макета
2. Нажмите на поле «Статус»
3. Выберите новый статус из выпадающего списка
4. Изменение сохраняется

### Связь со статусом в договоре

Руководитель и менеджер видят статус согласования макета в карточке договора. Обновляйте статус своевременно — это помогает всей команде видеть реальное состояние проекта.

## 8. История изменений

Раздел **«История»** — журнал ваших действий в системе.

**Что фиксируется:**
- Создание и редактирование макетов
- Смена статусов
- Загрузка и удаление изображений

Используйте историю для отслеживания хронологии работы над конкретным макетом.

---

## 9. Общий чат

Раздел **«Сообщения»** — общий командный чат.

**Как пользоваться:**
1. Откройте раздел «Сообщения»
2. Введите сообщение
3. Нажмите **Enter** или кнопку отправки

**Кто видит:**
- Все сотрудники студии (все роли)

**Используйте общий чат для:**
- Вопросов руководителю или менеджерам
- Уточнения деталей по конкретному проекту
- Объявлений и координации

## Типичные рабочие сценарии

### Сценарий 1: Получение нового задания

1. Руководитель или менеджер сообщает о новом проекте (в общем чате или лично)
2. Уточнить детали: месяц, тип мероприятия, требования по зонам
3. Создать новый **макет** в разделе «Дизайнер»
4. Заполнить метаданные: даты, сложность
5. Установить статус **«В работе»**

### Сценарий 2: Работа над макетом

1. Открыть нужный макет
2. По мере готовности загружать изображения по зонам
3. При необходимости уточнять детали в **общем чате**
4. При завершении — обновить статус на **«На согласовании»**

### Сценарий 3: Согласование с клиентом

1. Менеджер получил обратную связь от клиента
2. При необходимости доработок — внести изменения в изображения
3. После одобрения клиентом — обновить статус на **«Согласован»**
4. Передать в производство — обновить статус на **«Передан в производство»**

### Сценарий 4: Завершение работы

1. После мероприятия — обновить статус на **«Закрыт»**
2. Закрытые макеты остаются в архиве для справки

## Частые вопросы

**Где взять референсы для работы?**
Менеджер загружает референсные изображения в карточку договора. Попросите менеджера предоставить доступ к деталям — он может прислать файлы через общий чат.

**Как сообщить об изменении сроков?**
Напишите в **Общий чат** или обновите даты в карточке макета.

**Можно ли загрузить несколько изображений сразу?**
Да — при выборе файлов через диалог можно выбрать несколько файлов одновременно (зажав Ctrl или Shift).

**Что делать, если изображение слишком большое?**
Максимальный размер файла — 50 МБ. Сожмите изображение перед загрузкой (например, через онлайн-инструмент или Photoshop).

**Руководитель не видит мой статус. Почему?**
Убедитесь, что вы сохранили изменения статуса. Нажмите «Редактировать» → выберите статус → «Сохранить».
